# Bistutu空教室后端爬虫
​		该后端使用Java作为开发语言来编写，由于一些安全的原因，在爬虫中的API接口已经被删去，软件源码仅供学习交流之用。

## 服务器端使用到的第三方库

```
1. Maven 开发框架
2. fastJSON JSON数据解析
```
